# Australian Skills Shortage Dashboard — PostgreSQL + Power BI

## Overview
This project analyzes job postings, job seekers, shortages, and demand–supply ratios across key Australian occupations. 
The goal was to build a complete end‑to‑end analytics workflow using PostgreSQL for data storage and Power BI for visualization.

## Tools Used
- PostgreSQL
- Power BI
- SQL

## Dataset
The dataset includes:
- Occupation names
- Job postings
- Job seekers
- Shortage values
- Demand–supply ratios
- State information

## Database Setup
The SQL scripts used to create and populate the tables are included in the `sql_scripts` folder.

## Dashboard
The Power BI dashboard includes:
- Job postings by occupation
- Job seekers vs job postings
- Shortage by occupation
- Demand–supply ratio
- State-level slicer

## Key Insights
- Registered Nurses and Developers show the highest shortages.
- Demand–supply ratios highlight strong gaps in engineering and ICT roles.
- Job postings significantly exceed job seekers across all occupations.

## How to Use
1. Download the `.pbix` file.
2. Open in Power BI Desktop.
3. Connect to your PostgreSQL instance if needed.
